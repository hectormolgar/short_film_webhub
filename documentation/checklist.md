# Checklist de Entrega - Web-HUB Beta

Este documento certifica los estándares de calidad y funcionalidad para la entrega de la versión Beta operativa del proyecto cinematográfico.

## 1. Accesibilidad (WCAG 2.1)

- [x] **Contrastes**: El texto blanco/dorado sobre fondo negro cumple con los ratios de contraste recomendados para legibilidad.
- [x] **Semántica**: Uso de etiquetas correctas ( `<nav>`, `<header>`, `<main>`, `<footer>` ).
- [x] **Imágenes**: Atributos `alt` descriptivos incluidos en todas las tarjetas y elementos visuales.
- [x] **Escalado**: La interfaz permite zoom de hasta 200% sin pérdida de funcionalidad.

## 2. Diseño Responsive (Multidispositivo)

- [x] **Desktop**: Layout en grid de 3 columnas para visualización amplia.
- [x] **Tablet**: Adaptación de columnas y espaciados automáticos.
- [x] **Móvil**: Estructura de columna única y reducción dinámica de tipografía (clamped fonts).

## 3. Navegación y Estructura

- [x] **Enlaces**: Todos los enlaces internos (Inicio, El Corto, Equipo, Mapa Web) están operativos.
- [x] **HUB**: Página principal optimizada como punto de entrada (mission statement).
- [x] **Sitemap**: Página de mapa de contenidos incluida para facilitar el desarrollo futuro.
